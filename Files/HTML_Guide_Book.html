Chapter 01

HTML পরিচিতি

ভিডিও দেখুন

HTML কি?

HTML (HyperText Markup Language) হল ওয়েবের মেরুদণ্ড। এটি কোনো প্রোগ্রামিং ভাষা নয়, বরং একটি মার্কআপ ভাষা যা ওয়েব ব্রাউজারকে নির্দেশ দেয় ওয়েব পেজের কাঠামো কেমন হবে।

ইতিহাস

১৯৯১ সালে Tim Berners-Lee এটি আবিষ্কার করেন। বর্তমানে আমরা HTML5 ব্যবহার করি যা অনেক বেশি শক্তিশালী এবং আধুনিক।

কোড এডিটর সেটআপ

কোডিং করার জন্য ভালো এডিটর খুব জরুরি। নিচে সেরা কিছু অপশন দেওয়া হলো:

OS সেরা এডিটর ইনস্টলেশন পদ্ধতি
Windows VS Code code.visualstudio.com থেকে ডাউনলোড করে ইনস্টল করুন।
macOS VS Code / Sublime DMG ফাইল ডাউনলোড করে Applications ফোল্ডারে নিন।
Linux VS Code / Vim Terminal এ sudo snap install code কমান্ড দিন।

আপনার প্রথম কোড

index.html
<!DOCTYPE html> <html lang="bn"> <head> <meta charset="UTF-8"> <title>আমার প্রথম পেজ</title> </head> <body> <h1>হ্যালো বিশ্ব!</h1> <p>আমি HTML শিখছি।</p> </body> </html>

ট্যাগ বিবরণ ফলাফল

<p> এটি <strong>গুরুত্বপূর্ণ</strong> টেক্সট।
এটি <mark>হাইলাইট</mark> করা হয়েছে।

ব্লক লেভেল এলিমেন্ট (Block Level)

ভিডিও দেখুন

সেমান্টিক ট্যাগ কি?

সেমান্টিক ট্যাগগুলো কোডকে অর্থবহ করে তোলে। আগে আমরা সবকিছুর জন্য <div> ব্যবহার করতাম, কিন্তু HTML5 এ নির্দিষ্ট কাজের জন্য নির্দিষ্ট ট্যাগ আছে।

ট্যাগবর্ণনা
<header>ওয়েবসাইট বা সেকশনের উপরের অংশ।
<nav>ন্যাভিগেশন বা মেনুবারের জন্য।
<main>মূল কনটেন্ট বা প্রধান অংশ।
<section>নির্দিষ্ট কোনো সেকশন বা বিভাগ।
<article>ব্লগ পোস্ট বা আর্টিকেল।
<aside>সাইডবার বা অতিরিক্ত তথ্য।
<footer>নিচের অংশ বা ফুটার।
layout.html
<header> <h1>আমার লোগো</h1> <nav>...মেনু...</nav> </header> <main> <article> <h2>ব্লগ পোস্ট</h2> <p>বিস্তারিত লেখা...</p> </article> </main> <footer>কপিরাইট ২০২৫</footer>

মেটা ট্যাগ ও SEO

ভিডিও দেখুন

মেটা ট্যাগ কি?

মেটা ট্যাগ ব্রাউজার এবং সার্চ ইঞ্জিনকে পেজ সম্পর্কে তথ্য দেয়। এগুলো <head> এর ভেতরে থাকে এবং পেজে সরাসরি দেখা যায় না।

meta.html
<!-- বাংলা ফন্ট সাপোর্টের জন্য --> <meta charset="UTF-8"> <!-- মোবাইল রেসপন্সিভ এর জন্য --> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <!-- এসইও (SEO) এর জন্য --> <meta name="description" content="ফ্রি HTML টিউটোরিয়াল শিখুন"> <meta name="keywords" content="HTML, Bangla, Tutorial"> <meta name="author" content="Joy Kumhakar">

মনে রাখবেন:

ভালো র‍্যাঙ্কিং এর জন্য প্রতিটি পেজে সঠিক description মেটা ট্যাগ ব্যবহার করা অত্যন্ত জরুরি।

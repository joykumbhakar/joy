পাইথন কি?

Python একটি শক্তিশালী, হাই-লেভেল প্রোগ্রামিং ভাষা। এটি শেখা খুব সহজ কারণ এর সিনট্যাক্স ইংরেজির মতো। এটি ১৯৯১ সালে Guido van Rossum তৈরি করেন।

কেন পাইথন শিখবেন?

  • খুব সহজ সিনট্যাক্স।
  • ওয়েব ডেভেলপমেন্ট, ডাটা সায়েন্স, AI, হ্যাকিং সব ক্ষেত্রে ব্যবহৃত হয়।
  • বিশাল কমিউনিটি সাপোর্ট।

শর্তসাপেক্ষ বিবৃতি (Control Flow)

If, Elif, Else

প্রোগ্রামে সিদ্ধান্ত নেওয়ার জন্য আমরা শর্ত ব্যবহার করি। পাইথনে ইনডেন্টেশন (Indentation) বা স্পেস খুব গুরুত্বপূর্ণ।

marks = 85 if marks >= 80: print("A+") elif marks >= 70: print("A") else: print("Pass")

ইনডেন্টেশন সতর্কতা:

অন্যান্য ভাষায় {} ব্যবহার করা হয়, কিন্তু পাইথনে ৪টি স্পেস বা ১টি ট্যাব দিয়ে ব্লকের শুরু বোঝানো হয়। ভুল স্পেস দিলে IndentationError আসবে।

balance = 1000 while True: print("\n1. Check Balance\n2. Deposit\n3. Withdraw\n4. Exit") choice = input("Enter choice: ") if choice == '1': print(f"Your Balance: {balance} Tk") elif choice == '2': amount = int(input("Deposit Amount: ")) balance += amount print("Deposited Successfully!") elif choice == '3': amount = int(input("Withdraw Amount: ")) if amount <= balance: balance -= amount print("Withdrawn Successfully!") else: print("Insufficient Balance!") elif choice == '4': print("Thank you!") break
